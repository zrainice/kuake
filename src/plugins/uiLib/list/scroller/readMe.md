# scroller
## 组件说明
list下拉加载，下拉刷新

## 属性
| 参数   | 说明             | 类型    | 可选值 | 默认值 |
| ------ | ---------------- | ------- | ------ | ------ |
| noData | 是否可以下拉刷新 | boolean | -      | false  |

## 事件
| 名称     |      参数       | 说明     |
| -------- | :-------------: | -------- |
| refresh  | value, resolver | 下拉刷新 |
| infinite | value, resolver | 上拉加载 |
