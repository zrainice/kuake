# formRender
## 组件说明
```
用于 校验 表单属性控件必填对应信息，目前rules支持两种规则定义
1、required，如 rules: { 字段名：[{ required: true, message: '请上传图片' }]}
2、自定义规则，如 rules: { 字段名：[{ validator: (rule, value, callBack) => {
  // 拦击错误 callBack('error')
  // 成功 callBack()
} }]}
```

## 属性
| 参数  | 说明                     | 类型   | 可选值 | 默认值 |
| ----- | ------------------------ | ------ | ------ | ------ |
| model | 需要处理校验的表单值对象 | Object | {}     | {}     |
| rules | 需要处理校验的表单值规则 | Object | {}     | {}     |
